# Examples in _init_.py (in the form of Doc strings)

1) File Not Found Error:
- occurs when you are trying to access a file or folder that does not exist

2) Type Error:
- occurs when the data type of objects in an operation is inappropriate


Four Keywords for handling exceptions
1) try:
- defines a block of code test if it contains any errors.

2) except:
- defines a block of code to run if the try block raises an error

3) else:
- else block lets you execute code when there is no error

4) finally:
- finally block will be executed no matter if the try block raises an error or not

5) raise
- used to raise exceptions or errors.
